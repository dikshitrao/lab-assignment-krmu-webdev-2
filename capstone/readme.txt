## Project Overview
FoodieExpress is a fully responsive static website designed as a **Food Delivery App**.  
The project demonstrates modern front-end development techniques using **HTML5 semantic elements**, **CSS3 Flexbox & Grid**, and responsive design principles.  

The website includes a **Hero section, Menu, Featured Items, Contact form**, and a **Footer**, with interactive elements like hover effects, transitions, and simple animations.

---

## Learning Outcomes
By completing this project, students will be able to:

- Apply **HTML5 semantic elements** (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`) to structure web pages meaningfully.  
- Build **fully responsive layouts** using **Flexbox** and **CSS Grid**.  
- Implement **media queries** for different screen sizes (desktop, tablet, mobile).  
- Design visually appealing pages with **custom fonts, colors, and hover effects**.  
- Create functional **forms**, navigation menus, and interactive UI elements.  
- Organize and structure content for **user-friendly navigation**.  
- Apply **visual hierarchy and spacing** principles to improve readability.  
- Integrate **images, banners, and cards** effectively.  
- Gain experience in **full-page layout design** combining responsiveness, semantic structure, and modern CSS techniques.
