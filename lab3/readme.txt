## Project Overview
This project is a personal portfolio website developed as part of a *Web Design & Development Lab Assignment*.  
It demonstrates modern front-end techniques such as *Flexbox, **CSS Grid, **responsive design, **animations, and **hover effects*.  

The website includes the following sections: *Header, Hero, About, Skills, Projects, Contact, and Footer*.  
This project builds upon Lab 2, enhancing layout, responsiveness, and interactivity.

---

## Learning Outcomes
By completing this project, the student will be able to:

- Use *Flexbox* for alignment and layout control (Header, Hero section).  
- Implement *CSS Grid* for structured section layouts (Skills, Projects).  
- Create *responsive designs* using media queries for Desktop, Tablet, and Mobile.  
- Apply *relative units* (rem, %, vw, vh) for scalable spacing and fonts.  
- Add *hover effects* and *CSS transitions* for better user interaction.  
- Implement *keyframe animations* to make elements visually engaging.  
- Use *CSS transforms* for interactive elements like buttons or cards.  
- Maintain *design consistency* with spacing, alignment, and typography.
